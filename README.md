# monthly_streamflow_forecasting_lesotho
Datasets and Jupter notebooks for MSc dissertation on forecasting streamflow in Lesotho
# Streamflow and Climate Change in Lesotho

This repository contains datasets and Jupyter notebooks supporting the MSc dissertation:

**Forecasting Streramflow Dynamics in Lesotho using Machine Learning Methods**

## Repository Structure
- `data/` – CSV datasets (monthly_rainfall, monthly_temperature, monthly_streamflow_original, monthly_streamflow_imputed, cleaned_processed_data_scenario4, cleaned_processed_data_scenario5, final_merged_dataset)
- `notebooks/` – Jupyter notebooks for data cleaning, analysis, and modelling

## Data Coverage
- Period: 2001–2017
- Temporal structure: Monthly datasets

## Methods
Data-Driven methods utilised to predict monthly streamflow in the Malibamats'o river basin
